# Online Food Delivery Analysis

## Project Overview
This project analyzes online food delivery data to uncover insights on customer behavior, order trends, revenue, delivery performance, and restaurant performance. It involves data cleaning, feature engineering, exploratory data analysis (EDA), SQL analysis, and dashboard visualization.

## Folder Structure
- `data/` : Raw and processed datasets  
- `notebooks/` : Jupyter notebooks for data understanding, EDA, and feature engineering  
- `scripts/` : Python scripts for cleaning and preprocessing  
- `sql/` : SQL queries for analytics  
- `dashboard/` : Power BI or Streamlit dashboards  
- `app/` : Optional application or dashboard files  

## Key Features
- Data cleaning and preprocessing  
- Feature engineering (age groups, delivery performance, peak hours, profit margin %)  
- Exploratory Data Analysis (EDA) with plots and correlation analysis  
- SQL analytics for:
  - Top customers, high-revenue cities & cuisines  
  - Delivery performance and rating analysis  
  - Cancellation reasons and peak hour analysis  
- Dashboard showing:
  - Total Orders, Revenue, Average Order Value, Average Delivery Time  
  - Cancellation Rate, Average Delivery Rating, Profit Margin %  

## Tools & Libraries
- Python (pandas, numpy, matplotlib, seaborn)  
- MySQL for query-based analysis  
- Power BI or Streamlit for dashboards  

## How to Run
1. Open Jupyter notebooks to explore data and EDA.  
2. Run scripts in `scripts/` for preprocessing and feature engineering.  
3. Use SQL queries in MySQL Workbench for analytics.  
4. Open `dashboard/` Power BI or Streamlit project to visualize insights.
